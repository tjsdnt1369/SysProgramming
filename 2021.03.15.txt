.	: 현재 디렉토리 정보
..	: 부모 디렉토리 정보

cat	: 파일 읽기
cat > 파일명 -> 컨트롤 D	: 파일 생성

cd	: 디렉토리 변경
	ex) cd .. -> 부모 디렉토리로 이동
ls	: 현재 디렉토리 파일 목록을 보여준다 (list)
	-a 모든 파일 (숨김파일까지) 보여줌
	-l 세부정보 까지 길게 보여줌
cp 파일 경로	: 파일을 경로 안에 복사함
		(파일명을 같게, 같은 디렉토리에 복사하면 덮어쓰기 되고 파일명을 변경하면 새로운 복사파일이 생김)



멀티 시스템은 '나, group, others' 세가지로 구성 되어있다.
		나 자신, 자신이 속한 그룹을 제외한 모든것은 others
